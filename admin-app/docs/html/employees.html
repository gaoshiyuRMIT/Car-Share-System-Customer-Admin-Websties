Employee Routes¶

app.routes.employees.index()¶

defines index URL/route: ‘/’

Returns

html index

Return type

flask template

app.routes.employees.login()¶

Defines URL/route for login page: ‘/login’. Retrieves username and password from form, hashes password to compare with stored hash for entered username. If credentials are authorised, directs user to landing page depending on thier role (admin/manager/engineer).

Returns

html page (login/homepage) appropriate to user

Return type

flask template

app.routes.employees.logout()¶

Defines route for loggin out: ‘/logout’. Logs user out by clearing session details, and redirects to login page.

Returns

html for login page

Return type

flask template

app.routes.employees.menu()¶

Defines URL/route for main menu: ‘/menu’. All employees of type admin are directed to this page.

Returns

html for main menu page

Return type

flask template

app.routes.employees.register()¶

Defines URL/route for employee registration. Not currently active.

Returns

html page for registration

Return type

flask template
