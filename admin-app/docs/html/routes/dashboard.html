Dashboard Routes¶

app.routes.dashboard.car_usage()¶

Defines URL/route ‘/dashboard/data/car_usage’ which displays number of cars being used each day. Retrieves bookings data from cloud database and transforms for display in graph.

Returns

car usage data in json form

Return type

flask Response object

app.routes.dashboard.dashboard()¶

Defines URL/route for dashboard: ‘/dashboard’ which managers view on logging into employee website.

Returns

html page for dashboard

Return type

flask template

app.routes.dashboard.rental_revenue()¶

Defines URL/route ‘/dashboard/data/revenue’ which displays the daily revenue from renting cars. Retrieves booking history from cloud database and calculates amount of time cars have been rented for and resultant revenue.

Returns

revenue data in json form

Return type

flask response object

app.routes.dashboard.user_activities()¶

Defines URL/route ‘/dashboard/data/user_activities which displays number of active users each day. Information retrieved from cloud database table user_activity

Returns

number of active user/day is json form

Return type

flask response object
