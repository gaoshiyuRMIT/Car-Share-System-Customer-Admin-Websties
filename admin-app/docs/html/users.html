User Routes¶

app.routes.users.add_user()¶

Provides logic (via POST) to retrieve data entered on the add user page. Confirms entered username is unique, and prompts for different username if not. Data is then sent to cloud database to create a new record in user table.

Returns

confirmation message user was added, redirects to main menu

Return type

flask template

app.routes.users.add_user_page()¶

Defines URL/route for add car page, ‘/cars/add’.

Returns

html for add car form

Return type

flask template

app.routes.users.list_users()¶

Defines URL/route for displaying users. Retrieves users (stored in cloud database) and routes to html page: ‘/users’.

Returns

html page to display all users

Return type

flask template

app.routes.users.remove_user(username)¶

Defines route to remove user: ‘/users/username/remove’. Removes user from database and updates user list accordingly.

Parameters

username (string) – username=primary key for user in user table

Returns

updated list of users

Return type

flask template

app.routes.users.search_users()¶

Provides logic for filtering/searching displayed users according to their attributes/fields. Retrieves search terms from webpage and then matching users (from cloud database) and routes to ‘/users’.

Returns

html displaying all users matching search terms

Return type

flask template

app.routes.users.update_user(username)¶

Provides logic to retrieve edited details from update user form, and then insert in database. If password is changed, hashes password and stores hash. Redirects to list of users after update.

Parameters

username (string) – username=primary key for user in user table

Returns

Confirmation message, redirects to user view/search page.

Return type

flask template

app.routes.users.update_user_page(username)¶

Defines URL/route: ‘/users/username/update’ for updating the attributes/fields of a user (except username). Retrieves original user details from cloud database for display and editing.

Parameters

username (string) – username=primary key for users in user table

Returns

html for update user page

Return type

flask template
